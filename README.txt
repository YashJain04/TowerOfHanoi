Instructions to run:
1. Download folder and unzip
2. Please open up your terminal and change to the directory where you have downloaded the unzipped folder
3. Type "javac HanoiTower.java" to compile and create the classes
4. Type "java HanoiTower" to play the game with the classic 3 number of disks
5. Enjoy!
6. If you would like to challenge yourself, at Step 4 type "java HanoiTower X" where X represents any positive integer of your choice to play the game with a specific number of disks instead!
Example: "java HanoiTower 6" to play the game with 6 number of disks


The zip folder will contain the following files :

• Stack.java
• LinkedStack.java
• Player.java 
• HumanPlayer.java 
• GameState.java
• HanoiTower.java
• HanoiTowerGame.java
• AuthorInfo.java
• Utils.java
• README.txt

PROGRAM DESCRIPTION : 
This program is the implementation of HanoiTowerGame using the Java Programming Langauge.
The game was made using Stacks, LinkedStacks, and Interfaces.
